# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2024-02-01

### Features

- **ElCrud 主组件** - 一行配置生成完整 CRUD 页面
- **CrudTable** - 表格组件，支持列类型 (text, number, tag, datetime, image, link)、排序、多选、固定列
- **CrudSearch** - 搜索组件，支持字段类型 (input, select, date-picker, date-range-picker, time-picker, switch, checkbox, radio)、折叠展开
- **CrudDialog** - 弹窗表单组件，支持新增/编辑/查看模式、表单校验、自定义字段
- **CrudPagination** - 分页组件，支持自定义页码大小、布局配置
- **CrudToolbar** - 工具栏组件，支持新增、批量删除、刷新、密度选择、列设置
- **useCrud** - 组合式函数，封装完整 CRUD 逻辑
- **TypeScript** - 完整类型定义，支持 IDE 智能提示
- **Tree-shaking** - ESM/CJS 双格式输出，支持按需引入
- **单元测试** - 110 个测试用例，覆盖所有组件和组合式函数

# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2024-01-01

### Added

- Initial release
- CRUD table component with sorting, selection, and custom columns
- Search form component with multiple field types
- Dialog form component with validation
- Pagination component
- Toolbar component with create, batch delete, refresh, density, and column settings
- Full TypeScript support
- Custom slot support for all components
